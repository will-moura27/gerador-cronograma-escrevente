# 🚀 Meta Ciclo

O **Meta Ciclo** é uma plataforma de gestão de estudos voltada para concurseiros que buscam organização, clareza e progresso contínuo. Diferente de planilhas complexas, o Meta Ciclo foca na simplicidade do **ciclo de estudos**, permitindo que o usuário crie sua própria grade horária e acompanhe seu rendimento diário de forma intuitiva.

## ✨ Principais Funcionalidades

- **Gestão Dinâmica de Matérias:** Adicione as matérias do seu edital de forma rápida e simples.
- **Templates Prontos:** Carregue editais de concursos populares (como o TJ-SP/Vunesp) com apenas um clique.
- **Ciclos de Estudo Inteligentes:** O sistema calcula automaticamente o tempo disponível por matéria baseando-se na sua carga horária diária.
- **Acompanhamento de Progresso:** Barra de progresso visual para medir sua conclusão de tarefas.
- **Exportação para PDF:** Gere um relatório organizado do seu cronograma para imprimir ou consultar offline.
- **Experiência Nativa (PWA):** Instale o app diretamente na tela inicial do seu celular e acesse offline.

## 🛠 Tecnologias Utilizadas

Este projeto foi desenvolvido com tecnologias modernas de web:
- **HTML5 & CSS3:** Estrutura semântica e design responsivo.
- **JavaScript (ES6+):** Lógica de manipulação de dados e rodízio de matérias.
- **LocalStorage API:** Persistência de dados local (privacidade total do usuário).
- **Service Workers & Manifest:** Implementação PWA para acesso offline e instalação.
- **html2pdf.js:** Geração dinâmica de relatórios em PDF.
